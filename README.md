# Meta Tic-Tac-Toe
A recursive solution to the difficulty of tic-tac-toe. 

The game is written in **ES6**, **HTML5** and **CSS3**.

# Building
Building the project is done through [NW.js]() and [NWBuilder](). For ease of use, npm scripts have been included.

## Development
To run a development build, execute ``npm run dev``.

## Production
To produce a development build, execute ``npm run prod``. The builds will be saved into ``build``.
